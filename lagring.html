Lagring



Datorns tillfälliga minne mäts vanligtvis i gigabyte med GB-listan. Men det negativa är att minnet fungerar elektriskt, vilket innebär att strömmen inte kan sluta fungera. Om detta hände skulle det innebära att all information skulle gå förlorad. För att undvika fullt minne raderas information med jämna mellanrum. När programmet stängs kommer det omedelbart att tas bort från RAM-minnet. Detta bidrar i sin tur till att minnet sällan fylls, men den vanliga missuppfattningen är att människor tror att datorn blir föråldrad eftersom den åldras. Sanningen är ofta att RAM-minnet är fullt, program kan vara i bakgrunden och tar upp en betydande mängd minne, vilket i sin tur leder till en avmattning i datorn. RAM lagrar dock inte dokument som t.ex. bilder eller musik, precis som en hårddisk. Ju mer RAM en dator har, desto mer dator kan lagra med snabb åtkomst. Detta innebär att du kan ha många andra program på samma gång, utan att fylla minnet och bibehålla bra prestanda. Men å andra sidan måste du kontrollera hur mycket RAM som är begränsat. Denna begränsning beror på hur mycket information ditt moderkort kan hantera. För närvarande kan bärbara datorer inte hantera samma mängd RAM som stationära datorer. Minne kan mätas i KB, MB, GB.
